## OpenAI 的 Agents SDK 更新继续把企业代理开发推向标准化
- source: TechCrunch
- date: 2026-04-16
- category: 应用/产业
- url: https://techcrunch.com/2026/04/15/openai-updates-its-agents-sdk-to-help-enterprises-build-safer-more-capable-agents/
- summary: OpenAI 本周更新 Agents SDK，继续把企业代理开发从零散脚手架推向更标准化的产品路径。
- detail: |
    TechCrunch 报道，这次更新把更强的 sandbox、harness 和可控执行流程放到更核心的位置，让企业能在更熟悉的接口里构建长时程代理，而不必从头拼接隔离环境、状态管理和工具权限。

    这件事的重要性不只在 SDK 本身，而在于 OpenAI 正把“卖模型”延伸为“卖运行方式”。当越来越多客户真正让 agent 接文件、代码和内部系统时，标准化运行时就会变成企业采购决策的一部分。

- date: 2026-04-16
- category: 应用/产业

- detail: |
    新版 macOS 应用支持全局快捷键、屏幕理解、本地文件读取以及图像和视频生成，让 Gemini 从浏览器聊天框进一步变成可随时调起的桌面常驻助手。

## SiFive 的新融资说明 AI 基础设施争夺已下沉到服务器架构层
- source: TechCrunch
- date: 2026-04-16
- category: 基础设施
- url: https://techcrunch.com/2026/04/11/nvidia-backed-sifive-hits-3-65-billion-valuation-for-open-ai-chips/
- summary: SiFive 新一轮融资背后最值得关注的，不只是估值增长，而是 AI 基础设施竞争正继续下沉到 CPU 与服务器架构层。
- detail: |
    SiFive 的 RISC-V 路线试图为 AI 服务器提供更开放的 CPU 设计选择，并与 NVIDIA 的 CUDA 软件栈和 NVLink Fusion 方向形成新的组合，这让“开放 CPU + 封闭 GPU 生态”的混合模式更具现实感。

    过去大家更关心谁能买到 GPU，现在越来越多公司开始关心谁能参与定义整机。随着机架设计、功耗、散热和互连都成为瓶颈，底层架构话语权会变得越来越值钱。

## Capsule Security 证明运行时防护正在形成独立安全赛道
- source: VentureBeat
- date: 2026-04-16
- category: 安全
- url: https://venturebeat.com/business/capsule-security-exits-stealth-with-7m-to-stop-ai-agents-from-going-rogue-at-runtime
- summary: Capsule Security 带着新融资出场，显示围绕 agent 运行时防护的新安全赛道已经开始独立成形。
- detail: |
    这家公司主打在代理执行过程中实时监测异常工具调用、凭证误用和敏感数据外流，并在不重写现有代理框架的前提下，直接对运行时行为做约束和拦截。

    这个方向很有代表性，因为 agent 风险不再主要表现为“回答不安全”，而是“行动不安全”。一旦模型真正开始访问系统、调用接口和修改数据，运行时安全就会像 API 网关一样成为默认基础层。

## MIRAI 一类基准继续把研究重心从回答能力推向任务能力
- source: arXiv
- date: 2026-04-16
- category: 论文
- url: https://arxiv.org/abs/2407.01231
- summary: MIRAI 等基准持续受到关注，说明学界正把模型评估重心从回答能力转向任务能力。
- detail: |
    MIRAI 不是简单考语言模型会不会答题，而是要求 agent 在新闻、事件数据库和工具调用之间形成闭环，完成时序预测与信息整合，更接近真实工作任务。

    这类基准的价值在于，它迫使研究与产品一起面对更难的问题：模型不仅要会生成语言，还要会规划、检索、执行、校验和回溯。只要产品目标是“做成代理”，评测体系就迟早会向这里靠拢。

## 观察：4 月 16 日真正值得盯的是“谁同时控制入口与运行时”
- source: 综合观察
- date: 2026-04-16
- category: 观察,安全,基础设施
- url: https://www.anthropic.com/engineering/managed-agents
- summary: 从桌面入口到代理运行时，再到底层服务器架构，这几天的变化都在指向同一个问题：谁能同时控制入口与运行时。
