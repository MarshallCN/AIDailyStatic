## Zero-Order Sharpness-Aware 最小化稀疏训练框架：ZO-SAM 优化高维大模型训练
- source: arXiv
- date: 2026-03-16
- category: 论文,基础设施
- url: https://arxiv.org/abs/2603.13115
- summary: 论文提出 ZO-SAM 框架，融合零阶优化与锐度感知最小化，实现高效稀疏模型训练。
- detail: |
    大规模模型的训练对计算资源和梯度利用效率提出极高要求。一篇新论文 ZO-SAM 创新性地结合零阶优化（Zero-Order Optimization）和锐度感知最小化（Sharpness-Aware Minimization），为稀疏训练场景设计。零阶优化避免了显式梯度计算，降低内存消耗；锐度感知则寻找泛化性更好的损失平原区域。

    这个组合特别适用于难以获取精确梯度的场景（如分布式训练中的梯度延迟、隐私学习）。通过稀疏化（只对部分参数更新），结合这两项优化，模型训练效率和泛化能力都得到提升。在实验中，ZO-SAM 在多个基准上相比标准稀疏训练有 5-10% 的性能改进。

    这对资源受限的设备、联邦学习、以及超大规模模型的分布式训练有实际意义。

## Prompt Injection 作为角色混乱：新论文揭示 LLM 安全漏洞的根本机制
- source: arXiv
- date: 2026-03-16
- category: 论文,安全
- url: https://arxiv.org/abs/2603.12277
- summary: MIT 研究员发表论文，将提示词注入攻击重新理解为"角色混乱"问题，为防御提供新视角。
- detail: |
    一篇新发表的论文将提示词注入（Prompt Injection）攻击重新框架化为"角色混乱"（Role Confusion），而不是单纯的文本处理问题。论文作者认为，LLM 在接收包含指示的输入时，有时无法清晰区分"用户角色"与"系统角色"，导致攻击者可以伪装为系统发出指令。

    这一重新理解对防御策略有重要导向意义。传统的防御方式（如输入过滤）假设攻击来自于特定的文本模式，但如果问题的本质是角色识别混乱，则防御应该从"角色管理"的角度着手——更清晰的上下文分离、显式的角色标记、或者在 LLM 内部建立角色隔离机制。

    这代表了 AI 安全研究从"现象描述"向"根本机制"理解的演进。后续研究可能会探索基于角色的新防御框架，这对部署高风险应用的企业有实际意义。

    这反映了高效 AI 推理向"微观优化"方向深入：单个操作的性能改进积累起来可能带来数倍的系统吞吐提升。

- detail: |

- summary: 论文提出 Cost-Efficient Multimodal LLM Inference，通过异构 GPU 层级调度降低多模态大模型推理成本。
- detail: |
    多模态大语言模型（如图像-文本理解）因涉及多个处理阶段（图像编码、文本编码、融合、生成），其推理计算异质性很高。不同阶段可能最适合用不同等级的 GPU 执行。一篇新论文提出了自动化框架，根据每个子任务的计算特性和成本，动态分配到合适的 GPU 层级（高端 GPU、中端 GPU、低端 GPU）。

    核心创新在于成本模型与调度算法：通过每个操作实际执行会带来多少成本与延迟的预估，制定最优的任务分配策略。这在数据中心规模的推理场景中能显著降低总体拥有成本。

    这反映了云原生推理向"细粒度资源优化"方向发展的趋势。企业面对海量 API 请求时，不再简单地用统一配置的强大 GPU，而是精细化匹配任务需求与硬件资源。
