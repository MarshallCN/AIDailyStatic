## Anthropic 将 Claude Code 订阅与第三方接入分离
- source: TechCrunch
- date: 2026-04-04
- category: 应用/产业
- url: https://techcrunch.com/2026/04/04/anthropic-says-claude-code-subscribers-will-need-to-pay-extra-for-openclaw-support/
- summary: Anthropic 4月4日宣布 Claude Code 订阅从今天起不再覆盖 OpenClaw 等第三方接入需要额外按量付费。
- detail: |
    TechCrunch 报道，Claude Code 团队已在当地时间 4 月 4 日中午发出邮件，提醒订阅用户“无法再把 Claude 订阅额度用于 OpenClaw 及其他第三方前端”，必须通过和订阅分离的 pay-as-you-go 账单支付额外用量。

    Boris Cherny 解释这不是有意打击开放生态，而是订阅原本没为第三方支付模式设计，Anthropic 仍提供退款保障；OpenClaw 创始人 Peter Steinberger 也刚宣布加盟 OpenAI，但表示团队只争取到一周的缓冲期，未来必须为这些自由接入的工作负载另外掏钱。

## Hyunjin Kim 研究：AI 生产案例能迅速提升初创企业采用
- source: The Neuron
- date: 2026-04-04
- category: 论文
- url: https://www.theneuron.ai/explainer-articles/-around-the-horn-digest-everything-that-happened-in-ai-this-weekend-saturday-sunday-april-4-5-2026/
- summary: Hyunjin Kim 在 515 家高速成长初创公司中做的实地实验显示，展示 AI 生产案例能显著拉动实际使用、任务完成与收入，并减少对外部资本的依赖。
- detail: |
    这项实地实验由 Kim 及团队完成，给受测的 515 家高增长组约 44% 的公司展示了具体的 AI 生产化案例，结果是这些公司提出的 AI 使用场景数量提升 44%、完成的任务多出 12%、收入提升 1.9 倍，同时要求的外部融资比对照组少 39.5%。

    论文指出“Mapping Problem”依然是阻碍增长的关键摩擦——很多团队只要看到真实落地案例，就能立刻把 AI 项目映射到自己的业务；所以把这样高信噪、可复制的故事对外传播，能在没有额外模型能力投入的情况下加速整个生态的普及。

## 沙赫德无人机让波斯湾的 AWS 可用区大面积失联
- source: Gulf Insider
- date: 2026-04-04
- category: 基础设施
- url: https://www.gulf-insider.com/data-centers-hit-by-iranian-kamikaze-drones/
- summary: Gulf Insider 4月报道，三架伊朗 Shahed 无人机在 3 月袭击阿联酋与巴林的 AWS 数据中心，多个可用区的 EC2/S3/Lambda 同步脱离运行。
- detail: |
    这是首次确认的针对美国超级规模云设施的动能打击，文章指出那几座数据中心在 3 月被 Shahed 无人机击中后，不只墙体着火，还让 EC2、S3、Lambda 等核心服务断链，风控团队一但察觉就已经有银行、支付平台与叫车应用感受到链式断线。

    核心问题是这些超大数据中心本来只建成抗物理入侵与网络攻击的防线，“云就是一个个有围栏的建筑，少了空中防御”。文章建议用雷达、射频、EO/IR 侦测再配合主动拦截，甚至强调像 Sentradel 这样已在向数据中心推销自主反无人机系统；在波斯湾局势尚未缓和前，这类防护不能再拖。

- date: 2026-04-04

- summary: 周末摘要铺陈，一位自主 AI 智能体用四小时就绕过了某国最安全操作系统的硬化内核，暴露了防御带宽与自动化之间的巨大差距。

## x402 基金会在 Linux 基金会旗下启动支付层
- source: The Neuron
- date: 2026-04-04
- category: 生态
- url: https://www.theneuron.ai/explainer-articles/-around-the-horn-digest-everything-that-happened-in-ai-this-weekend-saturday-sunday-april-4-5-2026/
- summary: x402 基金会在 Linux 基金会的框架下曝光成立，携 Coinbase、Stripe、Visa、Google、Microsoft 等 20 多家厂商共同打造智能体铺开所缺失的支付层。
- detail: |
    周末特别专栏写到，x402 旨在把 agent 经济带与实际钱流彻底连通，成员名单涵盖 Coinbase、Stripe、Visa、Google、Microsoft 等，目标是建立一个标准化、可以信任的支付中介，让客户端/企业算力单元可以快速付费又不被单一平台捆绑。

    据说这个项目上线后 30 天就完成了 7500 万笔交易（文章是以 75 million transactions 计数），说明即使在 Early Stage，跨界联盟也能支撑起持续、可追踪的账务流，为未来将 agent 与真实世界合约绑定打下基础。

## Netflix 开源 void-model，去除视频中的对象与交互
- source: The Neuron
- date: 2026-04-04
- category: 开源
- url: https://www.theneuron.ai/explainer-articles/-around-the-horn-digest-everything-that-happened-in-ai-this-weekend-saturday-sunday-april-4-5-2026/
- summary: Netflix 发布 open-source void-model，通过视频语言模型把被因果影响的画面部位识别后准确删除，综合表现优于 ProPainter。
- detail: |
    Digest 介绍并附上 Hugging Face demo 链接，说明 void-model 是第一款视频对象与交互删除工具，利用 VLM 推理流水线识别哪些帧/对象在修改后会发生连锁反应，再决定怎么擦除或填补，实测比 ProPainter 更擅长维持画面一致性。

    Netflix 将源码开在 Hugging Face，同时给出的 demo 能直连 Sam 模型去精修，强调这个工具不是“封闭的擦除 API”，而是鼓励影片制作人、内容监管部门与开发者延伸用途，让视觉编辑从商业滤镜走向可解释的可逆操作。
