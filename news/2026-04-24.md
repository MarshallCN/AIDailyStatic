## GPT-5.5 和 GPT-5.5 Pro 开始进入 API，系统卡同步更新
- source: OpenAI
- date: 2026-04-24
- category: 应用/产业,生态,安全
- url: https://openai.com/index/introducing-gpt-5-5/
- summary: OpenAI 更新 GPT-5.5 发布说明，确认 GPT-5.5 和 GPT-5.5 Pro 已可用于 API，并补充系统卡安全信息。
- detail: |
    OpenAI 在 GPT-5.5 发布页中追加 4 月 24 日更新，说明 GPT-5.5 与 GPT-5.5 Pro 已进入 API，同时系统卡补充了适用于 API 部署的额外防护要求。

    这一步让 GPT-5.5 从 ChatGPT 和 Codex 的产品发布进入开发者与企业集成阶段。对应用团队来说，关键将变成如何在更强模型能力、成本、延迟和风险控制之间做部署取舍。

## BixBench 论文继续凸显科研 agent 的真实任务评测缺口
- source: arXiv
- date: 2026-04-24
- category: 论文
- url: https://arxiv.org/abs/2503.00096
- summary: BixBench 论文把 LLM agent 放进生物信息学数据分析流程，补上单轮问答评测难以覆盖的能力缺口。
- detail: |
    BixBench 设计了多步生物信息学任务，关注 agent 是否能探索数据、执行分析、解释结果并处理开放式问题。

    在 GPT-5.5 API 开放后，这类评测对开发者更有现实意义。模型进入 API 只是第一步，真正影响生产的能力是它能否在真实数据、工具调用和结果复核中保持可靠。
