## OpenAI 与 Microsoft 修改合作协议，Azure 优先但许可转向非独占
- source: OpenAI
- date: 2026-04-27
- category: 应用/产业,基础设施,生态
- url: https://openai.com/index/next-phase-of-microsoft-partnership/
- summary: OpenAI 与 Microsoft 公布修订协议，明确 Azure 优先、IP 许可到 2032 年且转为非独占，并调整收入分成安排。
- detail: |
    OpenAI 表示，修订后的协议为双方合作提供更长期的确定性，同时让 OpenAI 可以在任何云上向客户提供产品，但产品仍会优先在 Azure 上发布。

    这改变了 OpenAI 与 Microsoft 之间的约束结构。Microsoft 保留长期模型和产品许可，OpenAI 则获得更大的云和商业灵活性，双方关系从单一绑定走向更复杂的生态协作。
